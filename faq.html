Frequently Asked Questions

Everything you need to know about MoltBot Cloud.

🤖 General
What is MoltBot Cloud?
MoltBot Cloud provides pre-configured AI agent VMs for software development. Each VM runs 5-10 AI agents in parallel, powered by 12 LLM models with smart routing, persistent ChromaDB memory, and self-healing infrastructure. Think of it as having a full AI development team on a dedicated server.
How is MoltBot different from Cursor, Devin, or Copilot?
MoltBot runs multiple agents in parallel on a dedicated VM with persistent memory across sessions. Cursor and Copilot are single-agent IDE plugins. Devin is a single agent in a sandboxed environment at $500/mo. MoltBot gives you 5-10 agents with 12 models for $49-$299/mo. See the full comparison →
What can MoltBot agents do?
MoltBot agents can write code, run terminal commands, edit files, manage git repos, deploy to production, debug errors, write tests, generate documentation, and communicate via Telegram — all autonomously. They can work on multiple projects simultaneously.
💰 Pricing & Billing
How much does MoltBot Cloud cost?
Base: $49/mo (3-5 agents) · Swarm: $149/mo (5-10 agents) · Enterprise: $299/mo (10+ agents, priority GPU, 24/7 support). All plans include a 7-day free trial. View full pricing →
Is there a free trial?
Yes! All plans include a 7-day free trial with no credit card required. You get full access to all features during the trial period.
Can I change plans or cancel anytime?
Yes. Upgrade or downgrade anytime — changes take effect immediately. Cancel anytime with no penalties; cancellation takes effect at the end of your current billing period. Your data is preserved for 30 days after cancellation.
Do I need to pay for LLM API keys separately?
No. MoltBot includes smart model routing that uses free NVIDIA NIM models for simple tasks at zero cost. For premium models (Claude, GPT-4), usage is included in your plan up to your tier's limits.
🔒 Security & Data
Who can access my VM and code?
Only you. Each customer gets a dedicated, isolated VM. MoltBot staff cannot access your VM without your explicit permission. Your code never leaves your VM unless you deploy it.
Is my code used to train AI models?
No. We never use your code, prompts, or agent outputs to train AI models. Your data is yours. See our privacy policy for full details.
What happens to my data if I cancel?
Your data is preserved on the VM for 30 days after cancellation. You can download everything via RDP during that period. After 30 days, the VM is securely wiped.
🛠️ Technical
What OS do the VMs run?
Windows Server with full GUI access via Remote Desktop (RDP). All development tools, Node.js, Python, Git, VS Code, and the agent infrastructure come pre-installed.
Can I install my own software on the VM?
Yes! You have full administrator access. Install any IDE, framework, database, or tool you need. The VM is yours to customize.
What happens if a service crashes?
MoltBot includes self-healing infrastructure. Services auto-restart on failure, health checks run every 60 seconds, and you're notified via the status page. 99.5% uptime SLA. Check system status →
Can I use the API with my existing tools?
Yes. MoltBot exposes an OpenAI-compatible API at /v1/chat/completions. Any tool that works with OpenAI (LangChain, LlamaIndex, Continue.dev, etc.) works with MoltBot — just change the base URL. View API docs →
