API Reference

REST API for programmatic control of MoltBot agents, tasks, and infrastructure.

Authentication

All API requests require a Bearer token in the Authorization header. Generate API keys in your dashboard.

Authorization: Bearer mk_live_xxxxxxxxxxxxxxxxxxxx

Base URL

https://api.moltbot.cloud/v1

Agents

GET /agents List all agents

Returns all agents in your workspace with their status and resource info.

# Request
curl https://api.moltbot.cloud/v1/agents \
  -H "Authorization: Bearer mk_live_xxx"

# Response
{
  "agents": [
    {
      "id": "agent_01ABCDEF",
      "name": "coding-agent-1",
      "status": "running",
      "gpu": "rtx4090",
      "created_at": "2026-04-14T12:00:00Z",
      "memory_gb": 24,
      "active_task": "task_01XYZABC"
    }
  ],
  "total": 1
}
POST /agents Create a new agent
ParameterTypeRequiredDescription
namestringrequiredHuman-readable agent name
gpustringrequiredrtx4090 | a100 | h100
system_promptstringoptionalInitial system prompt for the agent
modelstringoptionalDefault LLM model (default: claude-opus-4)
 
curl -X POST https://api.moltbot.cloud/v1/agents \
  -H "Authorization: Bearer mk_live_xxx" \
  -H "Content-Type: application/json" \
  -d '{"name":"my-agent","gpu":"rtx4090","model":"claude-opus-4"}'
POST /agents/{id}/tasks Run a task on an agent
ParameterTypeRequiredDescription
taskstringrequiredNatural language task description
toolsarrayoptionalTool names to enable for this task
timeoutintegeroptionalMax seconds (default: 300)
 
curl -X POST https://api.moltbot.cloud/v1/agents/agent_01ABCDEF/tasks \
  -H "Authorization: Bearer mk_live_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "task": "Review the open PRs in github.com/myorg/myrepo and summarize each one",
    "tools": ["github", "web_search"],
    "timeout": 120
  }'

# Response
{
  "task_id": "task_01XYZABC",
  "status": "running",
  "estimated_completion_seconds": 45
}
DELETE /agents/{id} Stop and delete agent 
curl -X DELETE https://api.moltbot.cloud/v1/agents/agent_01ABCDEF \
  -H "Authorization: Bearer mk_live_xxx"

# Response 204 No Content

Python SDK

pip install moltbot
from moltbot import MoltBot

client = MoltBot(api_key="mk_live_xxx")

# Create and run an agent
agent = client.agents.create(name="my-agent", gpu="rtx4090")
task = agent.run("Audit our GitHub repo for security vulnerabilities")

# Poll for completion
result = task.wait()
print(result.output)

Node.js SDK

npm install @moltbot/sdk
import { MoltBot } from '@moltbot/sdk';

const client = new MoltBot({ apiKey: 'mk_live_xxx' });

const agent = await client.agents.create({ name: 'my-agent', gpu: 'rtx4090' });
const task = await agent.run('Generate unit tests for all untested functions');
const result = await task.wait();
console.log(result.output);
